Full-Stack blog website | React, Express, MongoDB, Nodejs
• Built a full-stack blog platform enabling users to create, read, and interact with blog
posts.
• Developed a responsive frontend using React and React Router for smooth navigation
and state management with hooks.
• Integrated Clerk for user authentication and session management.
• Created RESTful APIs with Express.js and Node.js for blog CRUD operations.
• Used MongoDB with Mongoose to model and store user profiles, blog content.
