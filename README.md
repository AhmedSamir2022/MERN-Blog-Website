Full-Stack Blog Website

A modern full-stack blog platform built with React, Express, MongoDB, and Node.js. Users can create, read, and interact with blog posts through a responsive interface, with secure authentication and efficient data management.

Features





User Authentication: Secure sign-up, login, and session management powered by Clerk.



Blog Management: Create, read, update, and delete (CRUD) blog posts via RESTful APIs.



Responsive Frontend: Built with React and React Router for seamless navigation and state management using hooks.



Backend APIs: RESTful APIs developed with Express.js and Node.js to handle blog and user operations.



Data Storage: MongoDB with Mongoose for modeling and storing user profiles and blog content.

Tech Stack





Frontend: React, React Router



Backend: Node.js, Express.js



Database: MongoDB, Mongoose



Authentication: Clerk
